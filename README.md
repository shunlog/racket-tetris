Wrote this to exercise the stuff I learned from [HtDP](https://htdp.org/),
and also to practice writing Lisp in Emacs.

You can install minimal Racket if you just want to try it out,
or you can get the full distribution to get the DrRacket IDE which has a REPL so you can play with the code.

To run the game, either run the `main.rkt` file from DrRacket, or from the console:
```
racket main.rkt
```
